Kids RX: Pediatric Electronic Medical Records

A full-stack application that allows providers to keep track of patient records. Like other EMR systems, features include appointment scheduling, creating patient charts, writing medical record notes, logging vital signs, and logging vaccination records. What makes Kids RX different is that it is specifically designed for use in pediatrics and includes features that were implemented with children in mind.

Created using HTML, Tailwind CSS, Bootstrap, custom CSS, JavaScript, Node.js, Express, Sequelize, Moment.js, Font Awesome, and Google Fonts.
